# 🎮 Hangman Game (Python)

## 📌 Project Description

This is a simple **Hangman Game** built using Python.
The player has to guess the hidden word (car brands) one letter at a time before running out of lives.

---

## 🎯 Objective

* Guess the correct word by entering letters
* You have **6 lives**
* Each wrong guess reduces one life
* Win by guessing all letters before lives reach zero

---

## 🧠 How It Works

* A random word is selected from a predefined list
* The word is hidden using `_ _ _`
* User inputs one letter at a time
* Correct guesses reveal letters
* Incorrect guesses reduce lives and display hangman stage
